# 三峰纸牌 (TriPeaks Solitaire)

## 1. 游戏概述

| 字段 | 值 |
|------|-----|
| 游戏 ID | `tripeaks` |
| 显示名称 | 三峰纸牌 |
| 难度 | ★★★☆☆ 中等 |
| 预计时长 | 3-8 分钟 |
| 玩家数 | 1 人 |
| 核心机制 | 连锁消除、递增/递减匹配 |

三峰纸牌是微软纸牌合集中最受欢迎的模式之一。三个重叠的金字塔（峰），玩家选择比基础牌大 1 或小 1 的牌消除，连续消除产生连锁得分加成。

### 1.1 游戏内规则展示

| 展示项 | 内容 |
|--------|------|
| **🎯 目标** | 消除 3 个金字塔共 30 张牌 |
| **🃏 布局** | 3 个峰，每峰 4 行 10 张 + 1 张基础牌 + 手牌堆 22 张 |
| **🎮 玩法** | ① 点击未被压住的牌 ② 该牌必须比当前基础牌大 1 或小 1 点 ③ 消除后变为新基础牌 ④ 连续消除产生连锁加分 ⑤ 无牌可消时翻手牌堆 |
| **✅ 点数循环** | A ↔ K ↔ Q ↔ ... ↔ 2 ↔ A（A 和 K 也相连）|
| **💡 技巧** | 规划消除顺序以最大化连锁；可消除的牌会自动高亮 |
| **🏆 胜利** | 30 张牌全部消除 |

## 2. 规则详情

### 2.1 牌具
- 标准 52 张扑克牌

### 2.2 布局

- **3 个峰**：每个峰是 4 行金字塔，共 10 张牌，3 峰共 30 张
- **手牌堆**：剩余 22 张

```
         ┌──┐         ┌──┐         ┌──┐
        ┌──┐┌──┐     ┌──┐┌──┐     ┌──┐┌──┐
       ┌──┐┌──┐┌──┐ ┌──┐┌──┐┌──┐ ┌──┐┌──┐┌──┐
      ┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐┌──┐
```

### 2.3 基本规则

**匹配：**
- 消除的牌必须比当前基础牌**大 1 点或小 1 点**（花色不限）
- 点数循环：A 和 K 相连（K→A 或 A→K 均可）

| 当前基础牌 | 可消除的牌 |
|-----------|-----------|
| 5 | 4 或 6 |
| A | K 或 2 |
| K | Q 或 A |

**流程：**
1. 点击可消除的牌 → 该牌变为新基础牌，连锁+1
2. 无可消时点击手牌堆翻一张 → 连锁重置

### 2.4 胜利条件
场上 30 张牌全部消除。

### 2.5 失败条件
手牌堆用尽且场上无可消除的牌。

## 6. 技术方案

### 6.1 核心算法

```javascript
const modDiff = (a, b) => {
  const diff = Math.abs(a - b);
  return diff === 1 || diff === 12;  // A↔K 循环
};
```
