# 时钟纸牌 (Clock Solitaire)

## 1. 游戏概述

| 字段 | 值 |
|------|-----|
| 游戏 ID | `clock` |
| 显示名称 | 时钟纸牌 |
| 难度 | ★☆☆☆☆ 纯运气 |
| 预计时长 | 1-2 分钟 |
| 玩家数 | 1 人 |
| 核心机制 | 幸运翻牌、时钟布局、仪式感 |

完全依靠运气的单人纸牌游戏。52 张牌排成时钟 12 个钟点 + 中心，玩家按顺序翻牌，结果完全由初始排列决定。胜利概率约 1/13。

### 1.1 游戏内规则展示

| 展示项 | 内容 |
|--------|------|
| **🎯 目标** | 成功翻开全部 52 张牌 |
| **🃏 布局** | 12 个钟点位置各 4 张（朝下）+ 中心 4 张（朝下），圆形排列 |
| **🎮 玩法** | ① 点击中心堆翻第一张牌 ② 看牌的点数放到对应钟点位置底部 ③ 从该钟点顶部取下一张翻牌 ④ 重复直到翻完或中断 |
| **✅ 点数映射** | A→1点, 2→2点, ..., 10→10点, J→11点, Q→12点, K→放回中心 |
| **💡 规则** | 完全随机，没有策略选择，纯靠运气 |
| **🏆 胜利** | 全部 52 张牌翻开（约 1/13 概率）|

## 2. 规则详情

### 2.1 点数与钟点对应

| A→1 | 2→2 | ... | 10→10 | J→11 | Q→12 | K→中心 |

### 2.2 布局

- **12 个钟点堆**：每个位置 4 张，正面朝下
- **中心堆**：4 张，正面朝下

```
           ┌──┐
           │11│
     ┌──┐  └──┘  ┌──┐
     │10│       │12│
   ┌──┐           ┌──┐
   │9│             │1│
 ┌──┐   ┌──────┐   ┌──┐
 │8│   │  中  │   │2│
 └──┘   │  心   │   └──┘
   └──┐  └──────┘  ┌──┘
   │7│             │3│
     └──┐         ┌──┘
     │6│  ┌──┐  │4│
       └──┘│5│└──┘
           └──┘
```

### 2.3 游戏流程

1. 从中心堆顶翻一张 → 看点数 → 放到对应钟点堆底部
2. 从该钟点堆**顶部**取一张翻 → 看点数 → 放到对应钟点堆底部
3. 循环直到某个钟点堆无牌可取

**胜利**：翻完 52 张（约 1/13 概率）。
**失败**：翻牌链中断（12/13 概率）。

## 6. 技术方案

### 6.1 游戏引擎

```javascript
function nextTurn() {
  const pile = piles[currentPile];
  if (pile.length === 0) return gameOver();
  const card = pile.shift();
  card.faceUp = true;
  revealed++;
  const hour = rankToHour[card.rank];
  if (hour === 'center') {
    piles[12].push(card);
  } else {
    piles[hour].push(card);
    currentPile = hour;
  }
  if (revealed === 52) return win();
}
```

### 6.2 发牌

```
1. seed → PRNG，洗牌
2. 第 1-4 张 → 1 点，5-8 → 2 点 ... 49-52 → 中心
3. 全部正面朝下
```
